## 🚀 Key Features
* **🎙️ Voice Enrollment:** Add medicines just by speaking (e.g., "Add Paracetamol at 8 PM").
* **🔔 Nagging Alert Logic:** A 3-burst voice reminder that repeats every 3 minutes until the medicine is marked as taken.
* **🤖 AI Medical Guide:** Ask questions like "What is Panadol?" to hear automated info on dosage and uses.
* **📱 Elder-First UI:** Modern "Glassmorphism" design with high-contrast colors and large, easy-to-read text.
* **🆘 SOS Protocol:** Emergency screen with one-tap dialing for quick help.
* **📊 Progress Tracker:** See exactly how many medicines were taken or missed for the day.

## 🛠️ How to Run
1. **Download:** Click the green **Code** button and select **Download ZIP**. Extract the folder.
2. **Setup:** Open your terminal or command prompt in that folder.
3. **Install:** Type `pip install -r requirements.txt` and press Enter.
4. **Launch:** Type `python final_app.py` and press Enter.

## ⚙️ Requirements
* Python 3.10+
* Working Microphone and Speakers
* KivyMD, Pyttsx3, and SpeechRecognition libraries
